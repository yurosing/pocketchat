# Регэкспы под сервер

Мод определяет ЛС по регулярным выражениям. По умолчанию они заданы под формат
**Essentials** вида «(ЛС) я → ник » текст». Если на вашем сервере другой формат —
поправьте паттерны.

::: warning Важно про группы
В паттернах ЛС и глобала: **группа 1 — ник, группа 2 — текст**.
:::

| Ключ | Что ловит |
|---|---|
| `incomingPattern` | Входящее ЛС (вам написали) |
| `outgoingPattern` | Исходящее ЛС (вы написали) |
| `globalPattern` | Строки публичного чата для вкладки «Глобал» |

## Значения по умолчанию

```json
"incomingPattern": "\\(ЛС\\)\\s*\\W*([A-Za-z0-9_]{2,16})\\s*->\\s*я\\s*»\\s*(.+)",
"outgoingPattern": "\\(ЛС\\)\\s*я\\s*->\\s*\\W*([A-Za-z0-9_]{2,16})\\s*»\\s*(.+)",
"globalPattern":   "([A-Za-z0-9_]{2,16})[^»A-Za-z0-9_]*»\\s*(.+)"
```

## Как подобрать под свой сервер

1. Отправьте себе тестовое ЛС и посмотрите **точный** текст строки в чате.
2. Замените в паттерне метку `(ЛС)`, стрелку `->` и разделитель `»` на те, что
   реально приходят на вашем сервере.
3. Оставьте группы `([A-Za-z0-9_]{2,16})` для ника и `(.+)` для текста.
4. Сохраните файл и перезапустите игру.

::: tip Экранирование в JSON
В JSON обратный слэш пишется дважды: `\\s`, `\\(`, `\\W`. Скобки-группы `(...)`
не удваиваются.
:::

## Каналы

Каналы (клан/альянс/группа) настраиваются в массиве `channels` и используют
**именованные группы** вместо нумерованных:

- `(?<name>…)` — ник;
- `(?<text>…)` — текст сообщения;
- `(?<clan>…)` — необязательный тег клана, показывается рядом с ником.

Пример канала «Клан» по умолчанию:

```json
{
  "id": "clan",
  "label": "Клан",
  "command": ".",
  "pattern": "^\\W{0,6}(?:клан|clan)\\W+.*?(?<name>[A-Za-z0-9_]{2,16})\\s*[»:>]\\s*(?<text>.+)"
}
```

Про использование каналов — в разделе [Каналы и группы](/guide/channels).
