# Внешний вид

<Shot src="/img/themes.png" caption="Настройка темы, цветов пузырей и обоев." />

| Ключ | По умолчанию | Описание |
|---|---|---|
| `theme` | `0` | 0 — тёмная, 1 — светлая |
| `uiScale` | `0` | Размер окна: 0 маленький, 1 средний, 2 большой |
| `fullscreen` | `false` | Полноэкранный режим (как Telegram Desktop) |
| `outColor` | `0` | Цвет своих пузырей (индекс палитры) |
| `inColor` | `0` | Цвет входящих пузырей (индекс палитры) |
| `uniformNames` | `false` | Единый цвет ников вместо «радуги» |
| `nameColor` | `0` | Индекс единого цвета ников |
| `msgTextColor` | `0` | Цвет текста сообщений (0 — авто) |
| `textScalePct` | `100` | Размер текста, % (60–150) |
| `badgeColor` | `0` | Цвет бейджа непрочитанных |
| `wallpaper` | `""` | Имя файла-обоев из `config/pmchat-wallpapers/` |

## Обои

Положите картинку в `config/pmchat-wallpapers/` и укажите её имя файла в `wallpaper`.
Пустое значение — без обоев.

## Цвета

Цвета задаются индексами по внутренней палитре мода. Проще всего подобрать их в
игре, на [экране настроек](/config/) — там видно превью.
