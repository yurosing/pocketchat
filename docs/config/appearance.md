# Внешний вид

Все эти настройки удобно менять прямо в игре, с превью — на экране настроек
PocketChat.

<Shot src="/img/themes.png" caption="Выбор темы, цветов пузырей и фона." />

| Настройка | По умолчанию | Что делает |
|---|---|---|
| Тема | Тёмная | Тёмная или светлая |
| Размер окна | Маленький | Маленький / средний / большой |
| На весь экран | Выключено | Окно занимает весь экран, как в Telegram Desktop |
| Цвет своих сообщений | — | Цвет пузырей, которые отправили вы |
| Цвет входящих сообщений | — | Цвет пузырей, которые прислали вам |
| Единый цвет ников | Выключено | Если включить — все ники одного цвета вместо разноцветных |
| Цвет текста сообщений | Авто | |
| Размер текста | 100% | Можно сделать крупнее или мельче |
| Цвет метки непрочитанного | — | |
| Фон переписки | Нет | Своя картинка вместо однотонного фона |

## Как поставить свой фон

Положите картинку в папку `pmchat-wallpapers` (где она находится — см.
[Где хранятся файлы мода](/reference/folders)), а затем выберите её в настройках
на экране «Фон переписки».

## Как подобрать цвета

Проще всего сделать это прямо в игре, на экране настроек — там видно, как
будет выглядеть переписка, ещё до сохранения.
