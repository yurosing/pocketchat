# Обзор pmchat.json

Все настройки лежат в файле `config/pmchat.json` и применяются **при следующем
запуске**. Большинство удобнее менять прямо в игре: **ModMenu → PocketChat →
Настройки** (или кнопкой-шестерёнкой в окне мессенджера).

<Shot src="/img/settings.png" caption="Экран настроек PocketChat в игре." />

## Основные параметры

| Ключ | По умолчанию | Описание |
|---|---|---|
| `msgCommand` | `"m"` | Команда сервера для личных сообщений |
| `payCommand` | `"pay"` | Команда для перевода денег из чата |
| `hideChatLines` | `false` | Прятать перехваченные строки ЛС из обычного чата |
| `globalPrefix` | `"!"` | Префикс при отправке в глобальный чат |
| `soundEnabled` | `true` | Звук при входящем сообщении |
| `notifySound` | `0` | Звук уведомления: 0 опыт, 1 колокольчик, 2 предмет, 3 выкл |
| `notifyVolume` | `100` | Громкость уведомления, % (5–100) |
| `dnd` | `false` | «Не беспокоить» — без всплываний и звука |
| `mentionEnabled` | `true` | Подсветка и пинг при упоминании вашего ника |
| `mentionExtra` | `""` | Доп. слова-триггеры через запятую |
| `mentionOnCopy` | `false` | Добавлять «ник: » в начало при копировании |
| `closeOnDamage` | `false` | Закрывать окно при получении урона |
| `enableMeta` | `true` | Индикатор набора и отметки о прочтении между модами |

## Разделы настроек

- [Внешний вид](/config/appearance) — тема, цвета, обои, масштаб.
- [Голос и озвучка](/config/voice) — Vosk и TTS.
- [Регэкспы под сервер](/config/patterns) — перехват ЛС, глобала и каналов.
- [Хостинги файлов](/config/hosts) — куда грузятся фото и голосовые.
- [Staff и CoreProtect](/config/staff) — функции для персонала сервера.

::: warning Правьте аккуратно
Файл — это JSON. Следите за кавычками и запятыми. Если файл повредить, мод при
запуске создаст новый со значениями по умолчанию.
:::
