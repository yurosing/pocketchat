# Настройки мода

Проще всего менять настройки прямо в игре — через **ModMenu → PocketChat** или
кнопкой-шестерёнкой в самом окне мессенджера. Всё, что описано ниже, можно
найти там же.

<Shot src="/img/settings.png" caption="Окно настроек PocketChat в игре." />

Технически все настройки хранятся в файле `pmchat.json` (где он лежит — см.
[Где хранятся файлы мода](/reference/folders)). Открывать этот файл руками
нужно, только если хотите что-то в нём точечно поправить — обычно это не
требуется.

## Основное

| Настройка | По умолчанию | Что делает |
|---|---|---|
| Команда личных сообщений | `m` | Какой командой мод отправляет ЛС на сервере |
| Команда перевода денег | `pay` | Какой командой отправлять деньги из чата |
| Прятать строки ЛС в обычном чате | Выключено | Убирает из обычного чата строки, которые мод уже показал в своём окне |
| Значок перед сообщением в общий чат | `!` | Символ, который мод сам добавляет при отправке в общий чат |
| Звук при новом сообщении | Включён | |
| Не беспокоить | Выключено | Отключает всплывающие уведомления и звук |
| Реагировать на упоминание ника | Включено | Подсветка и звук, когда кто-то написал ваш ник |
| Закрывать окно при получении урона | Выключено | |

## Разделы настроек

- [Внешний вид](/config/appearance) — тема, цвета, фон, размер окна.
- [Голос и озвучка](/config/voice) — распознавание речи и чтение чата вслух.
- [Как настроить под свой сервер](/config/patterns) — если сообщения не ловятся.
- [Куда грузятся файлы](/config/hosts) — фото, стикеры, голосовые.
- [Для персонала сервера](/config/staff) — функции для тех, у кого есть права модератора.

::: warning Осторожно с файлом настроек
Если решите открыть `pmchat.json` вручную — не удаляйте кавычки и запятые,
иначе файл перестанет читаться. Если что-то пошло не так — просто удалите файл,
мод создаст новый со значениями по умолчанию при следующем запуске.
:::
