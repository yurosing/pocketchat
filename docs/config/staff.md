# Staff и CoreProtect

Обе группы функций по умолчанию **выключены** и предназначены для персонала
сервера (хелперы и выше).

## Staff-функции (/warn)

Добавляет кнопку выдачи предупреждения прямо в ЛС и глобальном чате.

| Ключ | По умолчанию | Описание |
|---|---|---|
| `staffFeatures` | `false` | Включить кнопку `/warn` |
| `warnCommand` | `"warn"` | Команда выдачи предупреждения (без слэша) |
| `warnReason` | `""` | Причина по умолчанию (пусто — не добавлять) |

## CoreProtect

Отдельная вкладка для вывода плагина CoreProtect. Строки показываются как есть,
без перевода.

| Ключ | По умолчанию | Описание |
|---|---|---|
| `coreProtectEnabled` | `false` | Включить вкладку логов CoreProtect |
| `coreProtectPattern` | регэксп | Паттерн для перехвата строк плагина |

::: warning Не для общего доступа
Эти функции полезны только тем, у кого на сервере есть соответствующие права.
Обычным игрокам включать их смысла нет.
:::
