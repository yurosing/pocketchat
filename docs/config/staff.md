# Для персонала сервера

Эти две функции по умолчанию **выключены** и полезны только тем, у кого на
сервере есть права модератора или хелпера. Обычным игрокам их включать смысла
нет.

## Кнопка выдачи предупреждения

Добавляет кнопку выдачи предупреждения прямо в переписке и в общем чате — не
нужно вручную набирать команду.

Включается в настройках. Там же можно указать:
- какой командой сервер выдаёт предупреждения (по умолчанию `warn`);
- причину по умолчанию, если хотите, чтобы она подставлялась сама.

## Отдельная вкладка для логов CoreProtect

Если на сервере используется плагин CoreProtect (для отслеживания действий
игроков), можно включить для его вывода отдельную вкладку в мессенджере —
чтобы не искать нужные строки среди всего остального чата.

::: warning
Работает только если у вас есть права видеть вывод CoreProtect на сервере —
сам мод такие права не даёт.
:::
