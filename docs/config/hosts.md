# Куда грузятся файлы

Фото, стикеры, гифки и голосовые сообщения нельзя передать напрямую другому
игроку — Minecraft для этого не приспособлен. Поэтому мод сначала загружает
файл на один из бесплатных сайтов-хранилищ картинок, а собеседнику отправляет
только ссылку — так и работает большинство мессенджеров.

## Проверка, какие сайты сейчас доступны

Если фото не отправляются — введите в чат команду `/pm hosts`. Мод проверит
все сайты по очереди и покажет, какие из них сейчас работают с вашего
интернета.

::: tip
Некоторые сайты для загрузки картинок могут быть недоступны из-за
блокировок в вашей стране — это не поломка мода, просто конкретный сайт не
отвечает. Мод сам пробует следующий по списку.
:::

## Порядок, в котором мод пробует сайты

В настройках можно поменять, какой сайт пробовать первым (это может ускорить
отправку, если какой-то из сайтов у вас работает лучше остальных). По
умолчанию порядок такой:

1. kappa.lol
2. x0.at
3. qu.ax
4. catbox.moe *(заблокирован в России — стоит последним)*

Если, например, у вас стабильно работает только один из сайтов — можно
поставить его первым в списке в настройках.
