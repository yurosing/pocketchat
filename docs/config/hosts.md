# Хостинги файлов

Фото, стикеры, GIF и голосовые загружаются на публичные файлохостинги. Порядок
перебора задаётся в `uploadOrder` — коды через запятую, слева приоритетнее.

| Код | Хост | Примечание |
|---|---|---|
| `k` | kappa.lol | по умолчанию первый |
| `x` | x0.at | |
| `q` | qu.ax | |
| `c` | catbox.moe | заблокирован РКН — стоит последним |

Значение по умолчанию:

```json
"uploadOrder": "k,x,q,c"
```

## Проверка доступности

Команда `/pm hosts` перебирает все хосты и показывает, какие отвечают с вашего
интернета. Если фото не грузятся — начните с неё.

::: tip Переставьте порядок
Если, например, у вас стабильно работает только `x0.at`, поставьте его первым:
`"uploadOrder": "x,k,q,c"`.
:::

## Переопределение URL

В `hostOverrides` можно вручную переопределить URL скачивания по коду хоста —
на случай смены доменов или зеркал.

| Ключ | Описание |
|---|---|
| `uploadUrl` | URL загрузки (по умолчанию catbox API) |
| `imageHost` | База URL для скачивания по id |
| `hostOverrides` | Карта «код хоста → URL» для переопределения |
