# Скриншоты для документации

Клади сюда PNG-файлы с **точно такими именами**. Пока файла нет — на сайте
показывается стильная заглушка с точным именем ожидаемого файла. Как только
положишь файл с нужным именем и пересоберёшь (или обновишь dev-страницу) —
заглушка сама заменится картинкой.

Эта папка — `docs/public/img/`. В VitePress всё из `public/` доступно по
корню сайта, поэтому в разметке путь пишется как `/img/hero.png`.

## Как быстро вставить скриншот (Windows, Ctrl+V)

1. Сделай снимок: **Win + Shift + S** → выдели область (снимок копируется в буфер).
2. Открой **Paint** (Пуск → «Paint»).
3. Нажми **Ctrl + V** — картинка вставится.
4. **Файл → Сохранить как → PNG**, перейди в эту папку (`docs/public/img/`) и
   введи имя файла из таблицы ниже (например `hero`).
5. Готово. Если запущен `npm run docs:dev` — страница обновится сама.

> Совет: можно просто перетащить готовый PNG в эту папку, переименовав его в
> нужное имя.

## Список нужных файлов

| Имя файла          | Что должно быть на скрине                          | Где на сайте                   |
|--------------------|----------------------------------------------------|--------------------------------|
| `logo.png`         | Логотип/иконка мода (квадрат, ~200×200)             | Шапка и главная (hero)         |
| `hero.png`         | Всё окно мессенджера целиком (список + диалог)      | Стр. «Что это»                 |
| `mods-folder.png`  | Проводник: папка `mods/` с fabric-api и pmchat      | Стр. «Установка»               |
| `list.png`         | Левая панель — список диалогов с бейджами           | Стр. «Интерфейс»               |
| `context-menu.png` | Меню по правому клику на сообщении                  | Стр. «Интерфейс»               |
| `search.png`       | Результаты поиска по истории                        | Стр. «Интерфейс»               |
| `voice.png`        | Пузырь голосового с волной                          | Стр. «Голосовые сообщения»     |
| `stickers.png`     | Панель выбора стикеров/GIF                          | Стр. «Фото и стикеры»          |
| `global.png`       | Вкладка глобального чата                            | Стр. «Глобальный чат»          |
| `groups.png`       | Групповой чат / вкладки каналов                     | Стр. «Каналы и группы»         |
| `settings.png`     | Экран настроек в игре                               | Стр. «Обзор pmchat.json»       |
| `themes.png`       | Выбор темы, цвета пузырей, обоев                    | Стр. «Внешний вид»             |

Все скрины лучше делать шириной от ~1000px, формат PNG. Имена — строчными
латинскими буквами, ровно как в таблице.
