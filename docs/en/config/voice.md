# Voice & TTS

| Setting | Default | What it does |
|---|---|---|
| Speech recognition language | Russian | Russian or English |
| Read global chat aloud | Off | Global chat messages are read out with the system voice |

## How speech-to-text works

The first time you ask for a voice note to be turned into text, the mod
downloads a small file needed for recognition in the chosen language — Russian
or English. That's a one-time download, taking a bit of time depending on your
connection; after that, recognition works instantly with no internet needed.

If the main download source isn't reachable, the mod automatically tries a
backup — nothing for you to do, just wait a little longer the first time.

More on voice notes themselves in [Voice messages](/en/guide/voice).
