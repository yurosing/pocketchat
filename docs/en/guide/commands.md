# Commands & keys

## Keys

| Key | Action |
|---|---|
| <kbd>J</kbd> | Open or close the PocketChat window |
| <kbd>Ctrl</kbd>+<kbd>V</kbd> | Paste an image from the clipboard into the open chat |

::: tip Rebinding
The <kbd>J</kbd> key can be changed in **Options → Controls → Key Binds**
(PocketChat category).
:::

## Commands

| Command | Action |
|---|---|
| `/pm` | Open the PocketChat window (same as <kbd>J</kbd>) |
| `/pm hosts` | Test file host availability from your connection |

`/pm hosts` iterates over every host in [`uploadOrder`](/en/config/hosts) and
shows which respond from your connection. Handy when photos and voice don't
upload.
