# Commands & keys

## Keys

| Key | What it does |
|---|---|
| <kbd>J</kbd> | Open or close the PocketChat window |
| <kbd>Ctrl</kbd>+<kbd>V</kbd> | Send an image from the clipboard into the open chat |

::: tip You can rebind it
If <kbd>J</kbd> isn't convenient, go to **Options → Controls → Key Binds** in
Minecraft and assign your own (PocketChat category).
:::

## Chat commands

| Command | What it does |
|---|---|
| `/pm` | Open the PocketChat window (same as <kbd>J</kbd>) |
| `/pm hosts` | Check which photo/voice upload sites are currently reachable |

`/pm hosts` is handy when photos or voice notes won't send — it shows which
upload site is working from your connection. More in
[Where files get uploaded](/en/config/hosts).
