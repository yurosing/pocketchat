# Installation

PocketChat is a client-side **Fabric** mod. You install it only for yourself;
nothing needs to be installed on the server.

## Steps

1. Install [Fabric Loader](https://fabricmc.net/use/) 0.15 or newer for Minecraft **1.21.11**.
2. Download [Fabric API](https://modrinth.com/mod/fabric-api) and put the `.jar` into `mods/`.
3. Put `pmchat-mod-1.3.0.jar` into the same `mods/` folder.
4. Launch the game with the Fabric profile.

<Shot src="/img/mods-folder.png" caption="The mods/ folder with fabric-api and pmchat." />

::: warning Client only
Don't install the mod on the server — it works only on your side. Your contacts
don't have to install it, but with it they gain reactions, typing indicators and
read receipts.
:::

## Where the files live

After the first launch the mod creates its files in the `config/` folder of your
game instance. Details in [Folder layout](/en/reference/folders).
