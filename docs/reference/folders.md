# Где хранятся файлы мода

Все свои файлы мод складывает в папку игры, рядом с папкой `mods`, в подпапке
`config`.

## Как открыть эту папку

1. Нажмите <kbd>Win</kbd>+<kbd>R</kbd>, введите `%appdata%\.minecraft\config` и
   нажмите Enter.
2. Либо в лаунчере Minecraft: **Настройки запуска → Открыть папку игры**, а
   дальше зайдите в `config`.

## Что где лежит

```
.minecraft/
├─ mods/
│  └─ pmchat-mod-1.4.0.jar        # сам мод
└─ config/
   ├─ pmchat.json                 # все настройки
   ├─ pmchat-history.json         # вся ваша переписка
   ├─ pmchat-media/               # скачанные фото и голосовые
   ├─ pmchat-stickers/            # сюда кладёте свои стикеры и гифки
   ├─ pmchat-wallpapers/          # сюда кладёте картинки для фона переписки
   └─ pmchat-stt/                 # файлы для распознавания речи
```

| Папка или файл | Что там |
|---|---|
| `pmchat.json` | Все настройки — см. [Настройки мода](/config/) |
| `pmchat-history.json` | Вся переписка: диалоги, каналы, группы |
| `pmchat-media/` | Уже скачанные фото и голосовые (кэш) |
| `pmchat-stickers/` | Кладите сюда свои `.png` и `.gif` — станут стикерами |
| `pmchat-wallpapers/` | Кладите сюда картинки — станут доступны как фон |
| `pmchat-stt/` | Файлы для распознавания речи, скачиваются сами |

::: tip Резервная копия переписки
Чтобы не потерять историю сообщений, скопируйте файл `pmchat-history.json` в
надёжное место — например, при переустановке Windows.
:::

::: tip Секретные чаты сюда не попадают
[Секретные чаты](/guide/secret-chats) нигде не сохраняются — специально, чтобы
их нельзя было прочитать даже с доступом к этой папке. Они существуют только
пока открыта игра.
:::
