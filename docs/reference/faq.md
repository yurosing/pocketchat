# FAQ

## Входящие ЛС не появляются в моде

Формат ЛС вашего сервера не совпадает с регэкспом по умолчанию. Поправьте
`incomingPattern` / `outgoingPattern` — см. [Регэкспы под сервер](/config/patterns).

## Фото и голосовые не отправляются

Возможно, файлохостинги недоступны с вашего интернета. Проверьте командой
`/pm hosts` и переставьте порядок в [`uploadOrder`](/config/hosts).

## Собеседник должен ставить мод?

Нет. Он увидит обычный текст. Мод у обеих сторон нужен только для «мод-фич»:
реакций, статусов набора и отметок о прочтении.

## Где хранится моя переписка?

В файле `config/pmchat-history.json`. Только у вас, локально. См.
[Структуру папок](/reference/folders).

## Как переназначить клавишу открытия?

**Настройки → Управление → Клавиши** в Minecraft, категория PocketChat.
По умолчанию — <kbd>J</kbd>.

## Модель распознавания речи не скачивается

Основное зеркало могло быть недоступно. Мод пробует зеркала из `sttModelUrlRu` /
`sttModelUrlEn` по порядку. Можно указать своё зеркало — см.
[Голос и озвучка](/config/voice).

## Настройки не применились

Большинство настроек вступают в силу **при следующем запуске** игры. Перезапустите
клиент.

## Мод не виден в игре

Проверьте, что запущен профиль **Fabric**, а в `mods/` лежат и `fabric-api`, и
`pmchat-mod`. См. [Установку](/guide/install).
