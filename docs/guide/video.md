# Видео и YouTube

Видео можно смотреть прямо в мессенджере — не скачивая файл и не открывая
браузер. Работает и для своих видео, и для ссылок на YouTube.

<Shot src="/img/video-player.png" caption="Встроенный плеер: пауза, перемотка, громкость, скорость." />

## Что нужно для этого

Встроенный плеер работает через **VLC media player**, установленный у вас на
компьютере. У большинства он уже стоит; если нет — скачайте с
[videolan.org](https://www.videolan.org/vlc/). Без VLC мод просто откроет
видео по старинке — во внешнем плеере или браузере, ничего не ломается.

## Свои видео

1. Положите файл (`.mp4`, `.webm` и похожие) в папку `config/pmchat-video/`.
2. В диалоге нажмите кнопку ▶ (видео/аудио) и выберите файл — он загрузится и
   отправится собеседнику.
3. Кликните по видео в переписке — откроется плеер на весь экран.

## Ссылка на YouTube

Просто вставьте ссылку на видео (`youtube.com/...` или `youtu.be/...`) как
обычное сообщение и отправьте. Клик по ссылке в переписке открывает её во
встроенном плеере — так же, как своё видео.

::: tip Как это работает
Напрямую отдать YouTube-ссылку VLC уже нельзя — YouTube запирает потоки
защитой, которую обходит только специальный инструмент **yt-dlp**. Поэтому
при первом YouTube-видео PocketChat сам скачивает `yt-dlp` в
`config/pmchat-bin/` (плеер покажет «Получаю yt-dlp»), затем скачивает ролик
во временный файл («Скачивание N%») и играет его. Первый раз чуть дольше —
качается сам инструмент (~18 МБ), дальше только ролики.

Качество — до 720p: видео и звук качаются раздельными дорожками, а
встроенный плеер сводит их сам (ffmpeg не нужен). Если раздельно не вышло —
запасной вариант 360p одним файлом.
:::

::: warning Если пишет «Не получилось»
На некоторых роликах YouTube требует вход в аккаунт («подтвердите, что вы не
бот»). Тогда экспортируйте свои YouTube-куки в файл
`config/pmchat-cookies.txt` (формат Netscape — например, расширением браузера
«Get cookies.txt») — PocketChat подхватит их автоматически. Без кук такие
ролики открываются кнопкой «Открыть в браузере».
:::

## Управление в плеере

| Элемент | Что делает |
|---|---|
| ▶ / ❚❚ | Пауза и воспроизведение |
| Полоса перемотки | Клик или перетаскивание; при наведении — подсказка времени |
| Ползунок громкости | Клик или перетаскивание |
| Цифры скорости | Клик перебирает: 0.5x → 0.75x → 1x → 1.25x → 1.5x → 2x |
| Стрелка из рамки | Открыть исходную ссылку в браузере |
| Черта вверху (свернуть) | Свернуть в окошко в углу — чат остаётся рабочим |
| ✖ вверху или <kbd>Esc</kbd> | Закрыть плеер |

## Свернуть в окошко

Кнопка «свернуть» (черта) в правом верхнем углу плеера убирает видео в
маленькое окошко в углу экрана — при этом можно листать и писать в чат, а
видео продолжает играть. В окошке: клик по кадру — пауза, стрелки-рамка —
развернуть обратно, ✖ — закрыть.

Клик прямо по картинке видео тоже ставит на паузу/снимает с паузы.
