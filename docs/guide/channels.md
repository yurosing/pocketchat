# Каналы и группы

## Каналы (клан / альянс / группа)

**Каналы** — это отдельные чаты сервера, например чат клана или альянса.
Вкладка для такого чата появляется в мессенджере сама, как только мод впервые
увидит в нём сообщение. Из коробки уже настроены три:

| Вкладка | Как отправить сообщение |
|---|---|
| Клан | `.текст` |
| Альянс | `ally текст` |
| Группа | `gc текст` |

<Shot src="/img/groups.png" caption="Групповой чат и вкладки каналов." />

Если на вашем сервере другие названия чатов или команды — их можно поправить в
[настройках](/config/patterns#каналы).

## Группы

**Группы** — это чаты сразу с несколькими друзьями, которые вы создаёте сами,
прямо в моде. Технически мод просто рассылает обычные личные сообщения каждому
участнику по отдельности, а все ответы собирает в одну общую переписку — со
стороны сервера это выглядит как обычные `/m`.

::: tip
Группу видят одинаково все участники, у кого стоит PocketChat — переписка
собирается в одну ленту у каждого из них.
:::

## Паблики (публичные каналы)

**Паблики** — аналог Telegram-каналов: один владелец (и назначенные им
админы) публикует посты, остальные — подписчики без права письма, зато видят
число подписчиков и могут читать историю.

Создать канал — кнопка «＋ Новый канал» под списком групп. У владельца в
шапке канала есть кнопка ⓘ «О канале» — там код приглашения (скопировать и
отправить друзьям), список подписчиков с назначением админов и удаление
канала. Получатель вставляет код в «🔗 Войти по коду» — заявка уходит
владельцу автоматически.

::: tip
Владелец видит точное число подписчиков и просмотры под своими постами;
у подписчиков — последнее известное число. Канал можно закрепить-пост (ПКМ →
«Закрепить», доступно постящим) и заглушить уведомления колокольчиком в
шапке, как в Telegram.
:::
