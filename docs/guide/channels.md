# Каналы и группы

## Каналы (клан / альянс / группа)

**Каналы** — это серверные чаты. Вкладка появляется автоматически, как только мод
поймает первое сообщение канала по его регэкспу. По умолчанию заданы три:

| id | Вкладка | Команда отправки |
|---|---|---|
| `clan` | Клан | `.` |
| `ally` | Альянс | `ally` |
| `gc` | Группа | `gc` |

<Shot src="/img/groups.png" caption="Групповой чат и вкладки каналов." />

Каналы настраиваются в массиве `channels` файла `pmchat.json`. У каждого канала:

- `id` — внутренний идентификатор;
- `label` — название вкладки;
- `command` — команда отправки на сервере;
- `pattern` — регэксп с **именованными группами**:
  `(?<name>…)` — ник, `(?<text>…)` — текст, опционально `(?<clan>…)` — тег клана.

Подробнее про именованные группы — в разделе [Регэкспы под сервер](/config/patterns#каналы).

## Группы

**Группы** — локальные беседы на несколько игроков. Мод рассылает обычные `/m`
каждому участнику, а входящие собирает в одну ленту. Идентификатор группы
детерминированный (из состава участников), чтобы у всех участников с модом он
совпадал.

::: tip
Группы — чисто клиентская абстракция: сервер по-прежнему видит обычные личные
сообщения каждому участнику по отдельности.
:::
