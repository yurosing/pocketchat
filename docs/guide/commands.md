# Команды и клавиши

## Клавиши

| Клавиша | Действие |
|---|---|
| <kbd>J</kbd> | Открыть или закрыть окно PocketChat |
| <kbd>Ctrl</kbd>+<kbd>V</kbd> | Вставить изображение из буфера в открытый диалог |

::: tip Переназначение
Клавишу <kbd>J</kbd> можно сменить в **Настройки → Управление → Клавиши**
Minecraft (категория PocketChat).
:::

## Команды

| Команда | Действие |
|---|---|
| `/pm` | Открыть окно PocketChat (то же, что <kbd>J</kbd>) |
| `/pm hosts` | Проверить доступность файлохостингов с вашего интернета |

Команда `/pm hosts` перебирает все хосты из [`uploadOrder`](/config/hosts) и
показывает, какие из них отвечают с вашего соединения. Полезно, если не грузятся
фото и голосовые.
