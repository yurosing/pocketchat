# Быстрый старт

1. Зайдите на сервер и нажмите <kbd>J</kbd> — откроется окно PocketChat.
2. Напишите кому-нибудь ЛС обычной командой (`/m Ник привет`) или прямо из окна.
3. Проверьте, что входящие и исходящие ЛС появились в моде как пузыри.

::: tip Если ЛС не появляются
Формат ЛС вашего сервера отличается от стандартного. Это решается настройкой
регэкспов — см. [Регэкспы под сервер](/config/patterns).
:::

## Мини-чеклист

| Шаг | Как проверить |
|---|---|
| Мод загрузился | В **ModMenu** есть пункт PocketChat |
| Окно открывается | <kbd>J</kbd> открывает/закрывает мессенджер |
| ЛС ловятся | Входящее ЛС появляется отдельным диалогом |
| Фото работают | <kbd>Ctrl</kbd>+<kbd>V</kbd> в диалоге отправляет картинку |
| Хостинги доступны | `/pm hosts` показывает зелёные хосты |

## Что настроить в первую очередь

- [Регэкспы под сервер](/config/patterns) — если ЛС не ловятся.
- [Хостинги файлов](/config/hosts) — если не грузятся фото/голосовые.
- [Внешний вид](/config/appearance) — тема, цвета, размер окна.
