# Быстрый старт

1. Зайдите на сервер и нажмите <kbd>J</kbd> — откроется окно PocketChat.
2. Напишите кому-нибудь личное сообщение обычной командой (`/m Ник привет`) или
   прямо из открытого окна.
3. Проверьте, что и отправленное, и полученное сообщение появилось в окне
   пузырями.

::: tip Если сообщения не появляются
Скорее всего, на вашем сервере личные сообщения выглядят иначе, чем ожидает
мод по умолчанию. Поправить можно за пару минут — см.
[Как настроить под свой сервер](/config/patterns).
:::

## Проверка по шагам

| Шаг | Как понять, что всё ок |
|---|---|
| Мод загрузился | В списке модов (ModMenu) есть PocketChat |
| Окно открывается | Кнопка <kbd>J</kbd> открывает и закрывает окно |
| Сообщения ловятся | Полученное личное сообщение появляется отдельным диалогом |
| Фото работают | <kbd>Ctrl</kbd>+<kbd>V</kbd> в диалоге отправляет картинку |
| Сайты для фото доступны | Команда `/pm hosts` показывает хотя бы один рабочий |
