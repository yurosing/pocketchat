# Установка

PocketChat — клиентский мод для **Fabric**. Устанавливается только вам; на сервер
ничего ставить не нужно.

## Шаги

1. Установите [Fabric Loader](https://fabricmc.net/use/) 0.15 или новее для Minecraft **1.21.11**.
2. Скачайте [Fabric API](https://modrinth.com/mod/fabric-api) и положите `.jar` в папку `mods/`.
3. Положите `pmchat-mod-1.3.0.jar` туда же, в `mods/`.
4. Запустите игру профилем Fabric.

<Shot src="/img/mods-folder.png" caption="Папка mods/ с fabric-api и pmchat." />

::: warning Только клиент
Не ставьте мод на сервер — он работает лишь на вашей стороне. Собеседникам мод
устанавливать не обязательно, но с ним им станут доступны реакции, статусы набора
и отметки о прочтении.
:::

## Где находятся файлы

После первого запуска мод создаёт свои файлы в папке `config/` вашего экземпляра
игры. Подробно — в разделе [Структура папок](/reference/folders).
