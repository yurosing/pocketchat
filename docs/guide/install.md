# Установка

PocketChat ставится только вам на компьютер — на сервере ничего настраивать не
нужно. Всего 4 шага, если Minecraft уже с Fabric. Если ещё нет — ниже есть
разбор с самого начала.

## Что нужно скачать

| Что | Зачем | Где скачать |
|---|---|---|
| Fabric Loader | Позволяет ставить моды в Minecraft | [fabricmc.net/use](https://fabricmc.net/use/) |
| Fabric API | Мод, от которого зависят почти все другие моды, включая PocketChat | [modrinth.com/mod/fabric-api](https://modrinth.com/mod/fabric-api) |
| PocketChat | Сам мессенджер | у того, кто вам его прислал / со страницы мода |

Версия Minecraft — **1.21.11**.

## Если Minecraft ещё без Fabric (с нуля)

1. Откройте [fabricmc.net/use](https://fabricmc.net/use/) и скачайте **установщик** (Fabric Installer).
2. Запустите скачанный файл. В открывшемся окне:
   - вкладка **Client**;
   - версия Minecraft — **1.21.11**;
   - нажмите **Install**.
3. Установщик сам создаст в лаунчере Minecraft новый профиль с названием
   «fabric-loader-...». Больше в установщике ничего трогать не нужно, можно
   закрыть.
4. Откройте обычный **лаунчер Minecraft**, в выпадающем списке профилей сверху
   выберите профиль **Fabric** и один раз нажмите **Play** — так лаунчер создаст
   папку игры под этот профиль. После того как игра откроется в меню — можно
   закрыть, дальше уже ставим моды.

## Куда класть файлы модов

1. Откройте папку игры:
   - **Windows:** нажмите <kbd>Win</kbd>+<kbd>R</kbd>, введите `%appdata%\.minecraft`, нажмите Enter.
   - Или в лаунчере Minecraft: **Настройки запуска → Открыть папку игры**.
2. Внутри найдите (или создайте, если её нет) папку **`mods`**.
3. Перетащите туда скачанные файлы:
   - `fabric-api-*.jar`
   - `pmchat-mod-1.3.0.jar`

<Shot src="/img/mods-folder.png" caption="Папка mods/ с fabric-api и pmchat." />

::: tip Файл не запускается двойным кликом — это нормально
Файлы модов (`.jar`) не открываются сами по себе, как игры или программы. Их
не нужно ничего «устанавливать» — просто положите файл в папку `mods`, и всё.
:::

## Запуск

Откройте лаунчер Minecraft, выберите профиль **Fabric** вверху и нажмите **Play**.
Мод подключится сам — на сервере ничего настраивать не нужно.

::: warning Только для вас
Мод работает лишь на вашей стороне. Собеседникам его ставить не обязательно —
но если поставят, у вас появятся реакции, статус «печатает» и отметки о
прочтении.
:::

## Частые ошибки

### Мода нет в игре / ничего не изменилось

- Проверьте, что в лаунчере выбран именно профиль **Fabric**, а не обычный
  «Release» или «Latest release».
- Проверьте, что оба файла — и `fabric-api`, и `pmchat-mod` — лежат **прямо** в
  папке `mods`, а не во вложенной папке или архиве (файлы не должны лежать в
  `mods/mods/...` или оставаться внутри `.zip`).
- Убедитесь, что версия Fabric-профиля — **1.21.11**, как и версия модов.

### Игра не запускается / вылетает при старте

- Почти всегда причина — забыли положить **Fabric API**. PocketChat без него не
  заработает.
- Проверьте версию Minecraft у профиля Fabric — должна совпадать с версией мода.

### Не нашёл папку `mods`

Она не создаётся сама, пока вы хотя бы раз не запустите игру с Fabric-профилем.
Сначала один раз откройте игру профилем Fabric (можно сразу закрыть после
загрузки), потом создавайте папку `mods` вручную, если её всё ещё нет.

### Не вижу расширение `.jar` / файл выглядит по-другому

Если Windows скрывает расширения файлов, скачанный файл может отображаться без
`.jar` на конце — это нормально, главное чтобы файл был в папке `mods` как есть,
без переименования.
