# Голосовые звонки <Badge type="tip" text="NEW" />

Одна кнопка прямо в переписке — и собеседник получает приглашение в голосовой
канал, без ручного набора команд.

## Что нужно для этого

Звонки работают через мод **[Simple Voice Chat](https://modrinth.com/plugin/simple-voice-chat)**
— тот самый, которым многие уже пользуются для обычного голосового общения на
сервере. Важно: он должен быть установлен **на сервере**, где вы играете (не
только у вас на компьютере) — сама команда приглашения обрабатывается сервером.

Если Simple Voice Chat на сервере не установлен, кнопка звонка просто ничего
не сделает — ошибок и вылетов не будет.

## Как позвонить

1. Откройте переписку с человеком, у которого точно есть PocketChat.
2. Нажмите значок телефонной трубки в верхней панели чата.
3. Готово. Если вы ещё не состоите в голосовой группе, PocketChat сам создаст
   её (с вашим ником в названии) и, как только сервер подтвердит создание,
   отправит собеседнику приглашение. Дальше уже как обычно у Simple Voice
   Chat: собеседник увидит приглашение через сам этот мод.

## Как выйти из канала

Кнопки в PocketChat для этого нет — воспользуйтесь тем же способом, что и
всегда для Simple Voice Chat: командой `/voicechat leave` или его собственным
меню (по умолчанию открывается клавишей <kbd>V</kbd> — вкладка группы).

::: tip Технически
Команда `/voicechat invite <ник>` работает только когда вы уже в группе — сама
группа при инвайте не создаётся. Поэтому PocketChat сначала проверяет членство
через API Simple Voice Chat и при необходимости создаёт группу тем же сетевым
пакетом, что и родное меню мода, а затем шлёт официальный
`/voicechat invite <ник>`. Дальше всё делает сам Simple Voice Chat.
:::
