# Голосовые звонки <Badge type="tip" text="NEW" />

Одна кнопка вместо ручной возни с голосовыми группами: PocketChat сам
создаёт голосовой канал и добавляет туда вас и собеседника.

## Что нужно для этого

Звонки работают через мод **[Simple Voice Chat](https://modrinth.com/plugin/simple-voice-chat)**
— тот самый, которым многие уже пользуются для обычного голосового общения на
сервере. Важно: он должен быть установлен **на сервере**, где вы играете (не
только у вас на компьютере) — команды создания голосовой группы идут именно
через сервер.

Если Simple Voice Chat на сервере не установлен, кнопка звонка просто ничего
не сделает — ошибок и вылетов не будет.

## Как позвонить

1. Откройте переписку с человеком, у которого точно есть PocketChat.
2. Откройте статистику диалога (значок со столбиками) и нажмите
   **«Позвонить»**.
3. Готово — вы уже в голосовом канале. У собеседника всё произойдёт
   автоматически, без каких-либо действий с его стороны: как только придёт
   приглашение, его игра сама подключит его к тому же каналу.

## Как завершить звонок

Та же кнопка, теперь она называется **«Завершить звонок»** — нажмите, чтобы
выйти из голосового канала. Собеседник узнает, что вы вышли, но сам он
останется в канале, пока не выйдет тоже.

::: tip Без экрана «входящий звонок»
Как и секретные чаты, звонок подключается сразу, без отдельного шага
«принять» — чтобы не городить лишний экран поверх мессенджера.
:::
