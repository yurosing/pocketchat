# Интерфейс

Слева — список диалогов, контакты и закреплённые чаты. Справа — активная переписка
с пузырями сообщений.

<Shot src="/img/list.png" caption="Левая панель — список диалогов с бейджами непрочитанных." />

## Контекстное меню сообщения

Нажмите **ПКМ** по любому сообщению — откроется меню в стиле Telegram:

- **Ответить** с цитатой (можно выделить фрагмент текста).
- **Реакция** — эмодзи, синхронизируется с другими игроками с модом.
- **Копировать** текст.
- **Закрепить** — до 30 закреплённых на диалог.
- **Удалить**.

<Shot src="/img/context-menu.png" caption="Меню по ПКМ: ответ, реакции, копировать, закрепить, удалить." />

## Поиск

Мгновенный поиск по всей сохранённой переписке — по всем диалогам сразу.

<Shot src="/img/search.png" caption="Результаты поиска по истории." />

## Контакты и закрепы

- **Контакты** — избранные собеседники, закрепляются вверху списка.
- **Закреплённые сообщения** — как в Telegram, отдельно на каждый диалог.

## Кто с модом

Маленькая точка рядом с ником показывает, установлен ли PocketChat у собеседника —
так вы понимаете, дойдут ли до него реакции, статусы набора и отметки о прочтении.
