# Интерфейс

Слева — список ваших переписок и избранные собеседники. Справа — открытая
переписка с пузырями сообщений.

<Shot src="/img/list.png" caption="Левая панель — список переписок с меткой непрочитанного." />

## Меню по правому клику

Нажмите **правой кнопкой мыши** на любое сообщение — откроется меню:

- **Ответить** цитатой (можно выделить только часть текста).
- **Поставить реакцию** — эмодзи, собеседник с модом её увидит сразу.
- **Скопировать** текст.
- **Закрепить** — сообщение останется наверху переписки.
- **Удалить**.

<Shot src="/img/context-menu.png" caption="Меню по правому клику: ответ, реакции, копировать, закрепить, удалить." />

## Поиск

Ищет мгновенно по всей сохранённой переписке — сразу по всем диалогам.

<Shot src="/img/search.png" caption="Результаты поиска по истории." />

## Избранные собеседники и закрепы

- **Избранные** — можно закрепить нужных людей вверху списка, чтобы не искать
  каждый раз.
- **Закреплённые сообщения** — как в Telegram, отдельно в каждой переписке.

## Значок «есть мод»

Маленькая точка рядом с ником показывает, стоит ли у собеседника тоже
PocketChat — если да, ему будут видны ваши реакции и статус «печатает».
