# Глобальный чат

Закреплённая вкладка показывает публичный чат сервера теми же аккуратными
пузырями: цветные ники, кликабельные ссылки и поле ответа — можно не выходить из
PocketChat, чтобы написать в общий чат.

<Shot src="/img/global.png" caption="Глобальный чат сервера внутри мессенджера." />

## Префикс отправки

Отправка в глобал использует префикс из настройки [`globalPrefix`](/config/).
На многих RU-серверах это `!`. Если у вас общий чат по умолчанию (без префикса) —
оставьте поле пустым.

## Как ловятся строки

Публичные сообщения определяются регэкспом [`globalPattern`](/config/patterns).
Если ники или текст в глобале распознаются неправильно — поправьте этот паттерн.

## Озвучка

Сообщения глобального чата можно озвучивать системным голосом — настройка
[`ttsGlobal`](/config/voice).
