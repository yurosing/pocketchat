# Профили, роли и подарки

Новое в 1.8.3: у каждого игрока есть профиль — как в Telegram.

## Открыть профиль

- **ПКМ** по диалогу в списке слева, **или**
- кнопка **«☺»** в шапке открытого личного диалога, **или**
- меню **«⋮» → «Профиль»**.

## Что показывает профиль

- **Полный ник с префиксом и суффиксом** — так, как игрок выглядит на сервере
  (роль, клан, донат-тег и т.п.). Ник берётся из списка игроков (таб).
- **Аватар** (скин), статус **в сети / не в сети**.
- **День рождения** и **описание** — их можно задать в своём профиле («Мой
  профиль»).
- **Баланс** — только в своём профиле, рядом с ником (нужен серверный плагин +
  Vault).

## Должность определяется автоматически

Роль-должность **не выставляется вручную** — она определяется из ника
(префикс/суффикс). Если в нике есть один из значков или слов, рядом с ником
появляется бейдж:

| Значок | Должность |
|:---:|---|
| Ⓒ | Контент-мейкер |
| Ⓗ | Хелпер |
| Ⓜ | Модератор |
| Ⓔ | Ивент-мейкер |
| Ⓓ | Разработчик |

Также распознаются слова: `helper`/`хелпер`, `moder`/`модер`,
`developer`/`разраб`, `event`/`ивент`, `content`/`контент`.

## Переименовать игрока

В профиле любого игрока есть поле **«Имя»**. Впишите туда своё имя — и игрок
будет отображаться под ним в списке переписок и в шапке диалога (реальный ник
остаётся в заголовке профиля). При переименовании игрок добавляется в
**контакты**. На отправку сообщений это не влияет — команды `/m` уходят по
настоящему нику. Чтобы убрать переименование, очистите поле.

## Чёрный список

В профиле игрока есть кнопка **«В чёрный список»**:

- у заблокированного **скрывается аватарка** (даже когда он онлайн);
- в шапке диалога появляется значок **«⊘ ЧС»**;
- без серверного плагина блокировка дублируется в Essentials командой
  `/ignore`.

Убрать из ЧС — той же кнопкой в профиле.

## Подарки за монеты (Vault)

Раздел **«Подарки»** работает, когда на сервере стоит плагин **PocketChat** +
**Vault** и экономика (например EssentialsX).

- Откройте профиль **другого** игрока → раздел «Подарки» → **клик по подарку**.
- Цена спишется с вашего баланса Vault, получателю придёт уведомление.
- Полученные подарки видны в профиле игрока.
- Каталог подарков настраивается в `config.yml` плагина (секция `gifts:`).

Без плагина раздел показывает «доступно с плагином».

## Темы интерфейса

В настройках добавлены новые темы (тёмные и светлые): **Полночь**, **Nord**,
**Роза**, **Песок** — в дополнение к прежним Тёмной, Светлой и Графит.

## Аватарки групп

Групповому чату можно поставить свою картинку: положите изображение в
`config/pmchat-avatars/` и нажмите на аватарку в шапке группы — она будет
перебирать доступные файлы.
