# Голосовые сообщения

Зажмите кнопку микрофона, скажите что-нибудь (до 20 секунд), отпустите — готово,
сообщение отправлено. В переписке оно покажется волной звука с таймером.

<Shot src="/img/voice.png" caption="Голосовое сообщение с волной звука и распознанным текстом." />

## Превращение голоса в текст

Голосовое можно превратить в текст прямо в игре — без интернет-сервисов, всё
происходит у вас на компьютере. При первом использовании эта функция один раз
скачивает нужный файл — займёт пару минут в зависимости от интернета, дальше
всё работает без ожидания.

- Язык распознавания меняется в настройках: русский или английский.
- Если скачивание не пошло с первого раза — мод сам попробует другой источник.

::: tip
Подробнее про эти настройки — на странице [Голос и озвучка](/config/voice).
:::

## Озвучка общего чата

Можно включить, чтобы сообщения из общего чата сервера читались вслух голосом
Windows — полезно, если играете и параллельно занимаетесь чем-то ещё.
