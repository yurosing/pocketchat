# Голосовые сообщения

Зажмите кнопку микрофона, говорите (до 20 секунд), отпустите — сообщение
отправится. Внутри пузыря показывается волна и таймер.

## Распознавание в текст (Vosk)

Голосовые можно распознавать в текст офлайн-движком [Vosk](https://alphacephei.com/vosk/).
При первом использовании мод скачивает языковую модель в папку `config/pmchat-stt/`.

- Язык распознавания задаётся настройкой [`sttLang`](/config/voice) (0 — русский, 1 — английский).
- Ссылки на модели — в `sttModelUrlRu` / `sttModelUrlEn` (список зеркал через запятую,
  пробуются по порядку; GitHub-зеркало стоит первым).

::: tip Модель качается один раз
Модель весит десятки мегабайт и остаётся на диске. Если основное зеркало
недоступно — мод пробует следующее из списка.
:::

## Озвучка глобального чата (TTS)

Настройка [`ttsGlobal`](/config/voice) включает озвучивание сообщений глобального
чата системным голосом.
